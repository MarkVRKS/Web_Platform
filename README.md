## Описание проекта

WebPlatform — это веб-приложение, которое предоставляет пользователям возможность создавать персональные курсы по индивидуальным предпочтениям. 
На данный момент реализован базовый интерфейс, а также настроена база данных с функциями авторизации и регистрации.

## Цели проекта

- **Создание интуитивно понятного интерфейса**: Разработка фронтенда, который будет удобным и простым в использовании.
- **Настройка надежной базы данных**: Реализация системы авторизации и регистрации пользователей с использованием безопасных методов хранения данных.
- **Расширение функционала**: Постепенное добавление новых функций и улучшение существующих на основе отзывов пользователей.
- **Обеспечение безопасности**: Реализация мер по защите данных пользователей и предотвращению несанкционированного доступа.
